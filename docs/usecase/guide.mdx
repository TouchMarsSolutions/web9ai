# General Guide of AI Use Cases

## A few categories

1. Text 2 Text (`Txt2Txt`)
   1. Summarization (`Summ`)
   2. Text Generation and Continuation
   3. Text Classification
2. Text 2 Image (`Txt2Img`)
   1. Image 2 Image (`Img2Img`)
   2. Image 2 Text (`Img2Txt`)
3. Text 2 Voice (`Txt2Voice`)
4. Voice 2 Voice (`Voice2Voice`)
5. Text 2 Video (`Txt2Video`)
6. Text 2 Code

## Keynotes Breakdown

### `Txt2Txt`

- Use text prompt to generate text
  
- core usage includes:
  1. Summarization
  2. Text Generation and Continuation, such as 
     1. email
     2. story
     3. manual report
- the best products: 
  1. [OpenAI ChatGPT](https://ai.com/)
  2. [Google Bard](https://bard.google.com/)
  3. [Claude](https://console.anthropic.com/)
  4. [Poe](https://poe.com/)
  5. [AiTok](https://chat.aitok.us/) Totally FREE

### `Txt2Img`

- Use text to generate images
- the best products - only 2:
  - [MidJourney](https://www.midjourney.com/app/) - [paid plan](https://www.midjourney.com/account/)
  - open sourced [Stable Diffusion](https://stablediffusionweb.com/)
    - a few playing projects or sites:
      - [SD WebUI](https://github.com/AUTOMATIC1111/stable-diffusion-webui) open sourced
      - [Run Diffusion](https://app.rundiffusion.com?ref=web9ai15)
      - [TUSI.art](https://tusi.art/)

### `Txt2Voice`

- use text to generate voice
- popular open source projects:
  - [Bark](https://github.com/suno-ai/bark)
- Chinese:
  - [Baidu](https://ai.baidu.com/tech/speech/tts)
  - [XunFei](https://www.xfyun.cn/services/online_tts)

### `Voice2Voice`

- use one voice to generate another voice
- popular open source projects:
  - [so-vits-svc](https://github.com/svc-develop-team/so-vits-svc)
